# Cards

### ChoreOps Tile Card
I have only set this card up to save space on the main screen in the kitchen.  It is easy for the kids and wife to see who is doing what on the current day.

There is only one state mentioned and that is for "due", what is due right now.  Another screen/popup has Pending or In Progress

If you want to add due and in_progress as well then copy the filter: from the choreops_chore_row_left card.
